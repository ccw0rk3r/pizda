# AI-Powered Binary Options Signals

A modern web application that provides AI-generated trading signals for binary options trading. The platform features a beautiful, responsive design and is optimized for all devices.

## Features

- Modern, responsive UI with beautiful animations
- Secure access key system
- Real-time trading signals with entry points, stop loss, and take profit levels
- Interactive TradingView charts
- User profile with signal history
- Admin panel for user management
- Mobile-friendly design

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- TradingView Widget
- Local Storage for data persistence

## Getting Started

### Prerequisites

- Node.js 18.0 or later
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/binary-options-signals.git
cd binary-options-signals
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
